Everything you need to know about Afelu Guardian - from basics to advanced usage

Afelu Guardian is a sponsor-based platform that bridges the digital divide by providing secure access to AI tools and web content. Sponsors (typically diaspora communities) purchase access codes that users in restricted regions can redeem to access AI assistance, web content, and other digital tools through our Telegram bot. The system maintains complete anonymity between sponsors and end users.

Yes, safety and anonymity are our core principles:

  • Complete anonymity: No connection between sponsors and end users
  • Secure codes: Access codes work like digital gift cards
  • Encrypted communication: All bot interactions use Telegram's encryption
  • No personal data: We don't collect or store personal information from users
  • Offline sharing: Codes can be shared without internet connection

Using the Telegram bot is simple:

  1. Find the bot: Search for @AfeluBot on Telegram or click this link
  2. Start conversation: Click "Start" or send /start
  3. Redeem code: Send /redeem ET-XXXX-XXXX (replace with your actual code)
  4. Use services: Try commands like /gpt, /fetch, /translate, /news
  5. Check quota: Send /myquota to see remaining requests

The bot is available 24/7 and responds instantly to commands.

If your code isn't working, try these steps:

  • Check format: Codes must be in format ET-XXXX-XXXX (case-sensitive)
  • Verify spelling: Make sure you're typing the code exactly as received
  • Single use: Each code can only be redeemed once by one user
  • Bot verification: Ensure you're messaging @AfeluBot (not a fake bot)
  • Network issues: Try again in a few minutes if there's a connection problem

If the code still doesn't work, contact support@afelu.com with your code and error message.
