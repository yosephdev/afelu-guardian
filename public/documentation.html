  • Sponsors (Diaspora): Purchase access codes from anywhere in the world
  • Users (Local): Redeem codes to access AI and web content via Telegram bot

Troubleshooting

‚ùå "Code not found" error

  • Double-check the code format: ET-XXXX-XXXX
  • Make sure you're typing it exactly as received
  • Codes are case-sensitive
  • Each code can only be used once

ü§ñ Bot not responding

  • Check your internet connection
  • Try sending /start to restart the bot
  • Make sure you're messaging @AfeluBot (not a fake bot)
  • Wait a few seconds between commands

‚è∞ "Quota exceeded" message

  • You've used all your AI requests for this code
  • Contact your sponsor for a new code
  • Check remaining quota with /myquota
  • Web fetches have separate limits

üí≥ Payment issues

  • Payment is processed securely by Stripe
  • Codes are delivered instantly after successful payment
  • Check your email (including spam folder)
  • Contact support@afelu.com if codes don't arrive

üåê Website access issues

  • Use /fetch command for best results
