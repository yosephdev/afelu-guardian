Support & Troubleshooting

üîß Common Issues & Solutions

‚ùì Access Code Problems

  • Invalid Code: Check format ET-XXXX-XXXX (uppercase, with dashes)
  • Already Used: Each code works only once - contact sponsor for new code
  • Not Working: Ensure you're messaging @AfeluBot on Telegram
  • Need New Code: Contact your diaspora sponsor or support@afelu.com

ü§ñ Bot Response Issues

  • Slow Responses: Check internet connection, wait during peak hours
  • No Response: Send /start to restart bot session
  • Error Messages: Try rephrasing your request or contact support
  • Rate Limiting: Slow down requests if you see warning messages

üí≥ Credit & Quota Issues

  • Out of Credits: Use /myquota to check balance
  • Credits Not Added: Verify code redemption was successful
  • Unexpected Usage: Image generation uses 3 GPT credits
  • Need More Credits: Contact sponsor or purchase professional access

üì± Technical Problems

  • Telegram Issues: Update app to latest version
  • Mobile Problems: Works on Android, iPhone, and basic phones
  • Data Usage: Optimized for low bandwidth Ethiopian networks
  • App Crashes: Restart Telegram and try again
